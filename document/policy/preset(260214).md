# 🎯 사전 등록(Preset) 정책 - 부모/홈 케어 강화 버전 (260214)

**사전 등록(Preset)이란?**
부모나 교사가 일일이 앱을 선택하지 않아도 상황에 맞춰 설계된 잠금+출석 정책 패키지입니다.

---

## 1️⃣ 역할별 시나리오 분류

사용자의 역할(Role)에 따라 최우선으로 노출되는 사전 등록 카테고리가 달라집니다.

### 🏠 부모(PARENT)용 - 홈 케어 시나리오
가정 내 생활 습관과 학습 집중을 위한 프리셋입니다.

| 이름 | 설명 | 잠금 타입 | 효과 |
| :--- | :--- | :--- | :--- |
| **💡 집중 학습** | 집에서 집중해서 공부할 때 사용합니다. | APP_ONLY | 교육/도구 외 게임, 소셜, 엔터 차단 |
| **🍱 식사 시간** | 식사 중 스마트폰 사용을 멈추게 합니다. | FULL | 모든 앱 차단 (전화 제외) |
| **🌙 취침 모드** | 자녀의 수면을 위해 폰 사용을 제한합니다. | FULL | 모든 앱 차단 (수면 유도 앱만 허용 가능) |
| **📺 유튜브 제한** | 과도한 영상 시청을 잠시 멈추고자 할 때 사용. | APP_ONLY | 엔터테이먼트 카테고리 집중 차단 |
| **🏃‍♂️ 자유 보상** | 할 일을 마친 후 보상으로 일부만 차단 해제. | APP_ONLY | 게임만 차단하고 나머지 허용 |

### 🎓 교사(TEACHER)용 - 학교/기관 시나리오
수업 운영 및 학원 출석 관리를 위한 프리셋입니다.

| 이름 | 설명 | 잠금 타입 | 효과 |
| :--- | :--- | :--- | :--- |
| **🏫 수업 집중** | 수업 중 스마트폰 사용을 방지합니다. | APP_ONLY | 교육용 앱만 사용 가능 |
| **📝 시험 모드** | 시험 시간 동안 부정행위를 방지하고 집중 유도. | FULL | 전체 앱 차단 |
| **📚 자율 학습** | 자습 시간 동안 학습 도구만 허용합니다. | APP_ONLY | 사전, 메모, 계산기 등 도구 앱 허용 |
| **✅ 출석 체크** | 잠금 없이 출석 정보만 수집합니다. | NONE | 출석만 기록 (잠금 없음) |

---

## 2️⃣ 스키마 확장 (Category 도입)

각 사전 등록은 `category` 필드를 가져 API 조회 시 필터링이 가능하도록 설계합니다.

- `HOME`: 부모 전용
- `SCHOOL`: 교사/기관 전용
- `COMMON`: 공통 (예: 시험 모드, 취침 시간)

---

## 3️⃣ UX 적용 원칙

1. **상황 기반 제목**: "어떤 상황인가요?"라는 질문에 바로 답할 수 있는 직관적인 이름 사용.
2. **미리보기**: 카드를 선택하면 어떤 카테고리가 차단/허용되는지 아이콘으로 명확히 인지.
3. **즉시성**: 부모는 여러 설정 없이 '밥 먹자!' 카드 클릭 후 QR 생성만으로 즉시 잠금 실행.
